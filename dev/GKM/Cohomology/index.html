isGKMclass(c::FreeModElem{QQMPolyRingElem}, G::AbstractGKM_graph) -> Bool

Return true if the given class is an actual GKM cohomology class. This holds if and only if the difference between localizations at fixed points connected through an edge e is divisible by the weight of e.
